# Minimal Version List Design

## Goal

Remove the card-like, overly decorative treatment from the dark version
workbench and make its version area read like a compact editor sidebar.

## Version List

- A version is one borderless list row with version, title, timestamp, and a
  small text apply action.
- The current version uses only a thin blue leading rule and a subtle surface
  tint. It does not use a card head, pill, or large action button.
- Status is represented by plain secondary text rather than a badge.

## Panel Tone

Keep the dark workbench surface but reduce rounded containers, heavy borders,
and decorative elevation. Existing drag, collapse, explicit apply, and status
behavior remain unchanged.
